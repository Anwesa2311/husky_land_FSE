Emoji_menu

emoji

This struct represents a emoji menu that will be rendered when the user enters the emoji sharing mode. There will be a total of five different emojis for each user to choose. A number that representing the code of the emoji will be rendered at the top of each emoji.

struct Emoji_menu {
Emoji_Option[5] emojiArray;
int window_width;
int window_height;
this(SDL_Renderer* renderer, int w, int h);
void Render(SDL_Renderer* renderer);
}
